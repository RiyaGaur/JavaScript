Arrays And Objects

Arrays

  • isArray(): determines whether an object is an array.
  • indexOf(): searches the array for the specified item, and returns its position.
  • push(): adds a new element to an array (at the end).
  • pop(): removes the last element from an array.
  • shift(): removes the first array element and "shifts" all other elements to a lower index.
  • unshift(): adds a new element to an array (at the beginning), and "unshifts" older elements.
  • splice(): remove elements without leaving "holes" in the array.
  • reverse(): reverses an array in place.
  • concat():creates a new array by merging (concatenating) existing arrays.

Objects

JavaScript objects are containers for named values called properties or methods.
Objects are in key:value pair.
