Variables: let, const & var in JavaScript

Rules for creating Variables:

  • Cannot start with numbers
  • Can start with letter, _ & $
  • Are case sensitive

Var And Const

Var can be modified while const is not.
Eg:

var name = 'Riya';
name = 'Riya Gaur';     ->Allowed

const headname = 'Riya';
headname = 'Riya Gaur';     ->Not Allowed



const arr=[12,23,53,3];
console.log(arr);
arr.push(43);      ->Allowed
console.log(arr);
arr=[11,12,13];      ->Not Allowed

Let

Variables declared with the let keyword can have Block Scope.

Variables declared inside a block {} cannot be accessed from outside the block.

Eg:

var city = 'Mathura';
{
     let city = 'Kolkata';
}

If we print the city outside the block then it will print 'Mathura' and if we print inside the block then it will print 'Kolkata'.
