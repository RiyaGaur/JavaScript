Data Types in JavaScript

Types of Datatypes

  1. Primitive Datatype
  2. Reference Datatype(Memory-Heap)

Primitive Datatypes:

  • String: " "
  • Numbers: 34
  • Boolean: true or false
  • Null
  • Undefined
  • Symbol: ES6

Reference Datatypes:

  • Arrays
  • Object literals
  • Functions
  • Dates

Important

  1. typeof is used to check the datastype of the variable.
  2. typeof null return object while it is primitive datatype
  3. Arrays, Objects, Dates return object
